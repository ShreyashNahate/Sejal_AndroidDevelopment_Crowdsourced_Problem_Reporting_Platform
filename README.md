# 🏙️ Crowdsourced Problem Reporting Platform

---

## 📌 Problem Statement (PS)

Urban areas frequently face civic issues such as potholes, garbage overflow, and water leakage. However, existing systems for reporting these problems are inefficient due to lack of transparency, delayed responses, and absence of a centralized platform.

Additionally, challenges like fake complaints, lack of prioritization, and poor citizen engagement make the system unreliable and ineffective.

---

## 💡 Solution Overview

FixMyCity is a Flutter-based mobile application that enables citizens to report civic issues in real-time using images and location data.

The system enhances traditional complaint mechanisms by introducing:

* 📸 Image-based issue reporting
* 📍 Automatic location detection
* ⭐ Trust-based weighted voting system
* 🤖 AI-powered chatbot for guidance
* 🕵️ Anonymous complaint reporting
* 🚨 Emergency mode for critical issues
* 🌍 Civic health map with gamified rankings

This ensures transparency, faster resolution, and improved citizen participation.

---

## 🚀 Key Innovations

* **Weighted Voting System:** Assigns trust scores to users to prevent fake voting
* **Smart Reporting:** Chat + voice-based complaint submission with auto-category detection
* **Fake Complaint Detection:** Multi-layer validation using location, images, and user behavior
* **Anonymous Reporting:** Protects user identity while maintaining system integrity
* **Emergency Mode:** Prioritizes urgent complaints with instant alerts
* **Gamified Civic Map:** Visualizes issue density and ranks cities to encourage improvement

---

## 🛠️ Tech Stack Used

* **Frontend:** Flutter (Dart)
* **Backend:** Firebase Firestore
* **Maps:** OpenStreetMap (via flutter_map)
* **Voice Input:** speech_to_text package
* **Authentication:** Optional / Dummy-based system

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/ShreyashNahate/Sejal_AndroidDevelopment_Crowdsourced_Problem_Reporting_Platform.git
cd smartcity
```

2. Install dependencies:

```bash
flutter pub get
```

3. Set up Firebase:

* Create a Firebase project
* Add Android app
* Download `google-services.json`
* Place it in `android/app/`

4. Run the app:

```bash
flutter run
```
